Описание предметной области

Онлайн магазин компьютерных игр. Продажа только зарегистрированным пользователям. Для регистрации достаточно почты, пароля. Почта нужна для подтверждения регистрации и отправки чеков об оплате. На каждого пользователя приходится свой список игр, которые он хотел бы купить. После покупки корзина обнуляется. На один аккаунт может быть привязан только уникальный набор игр. У каждой игры может быть несколько категорий.

Для большей отчетности, требуется хранить все оплаты по тем ценам, по которым они были совершены.

Пользователь может удалить свой аккаунт. Тогда его корзина удаляется.
